## 実装した内容

- 任意のユーザーが〇×クイズを作成できる
- 任意のユーザーが〇×クイズに回答し正誤を知ることができる

## デプロイ先の URL または画面収録したファイル名

hogehoge.mp4

## 使い方

1. Quizを作る
2. 自他のユーザーが作ったクイズに回答する

## 工夫した点

- 〇×のどちらかしか回答できないこと
- 回答者の答えと正答を照らし合わせて正誤を判定するところ
